# Mordell 方程的代数数论求解方法解析

本文档基于论文《Formalized Class Group Computations and Integral Points on Mordell Elliptic Curves》中的定理 5.1（Theorem 5.1），详细解析了如何利用代数数论中的**理想下降法（Ideal Descent）**和**类群（Class Group）性质**，来显式求解特定条件下的 Mordell 方程 `y² = x³ + d`。

---

## 1. 求解前提条件

论文主要考虑了满足以下特定条件的 `d` 值：
1.  **`d < 0`**：`d` 是一个负整数。
2.  **无平方因子（Squarefree）**：`d` 不能被任何素数的平方整除。
3.  **模 4 同余条件**：`d ≡ 2 (mod 4)` 或 `d ≡ 3 (mod 4)`。这个条件确保了二次域 `ℚ(√d)` 的整数环 `D` 恰好是 `ℤ[√d]`。
4.  **类数条件（核心）**：二次环 `D = ℤ[√d]` 的类数 `h(D)` **不能被 3 整除**（即 `3 ∤ h(D)`）。

---

## 2. 核心求解步骤

求解过程是一个非常优雅的纯代数推导，主要分为以下六个步骤：

### 步骤一：在二次环中因式分解
首先，将原方程 `y² = x³ + d` 移项为 `y² - d = x³`。
然后，在二次整数环 `D = ℤ[√d]` 中对左边进行因式分解：
```text
(y + √d)(y - √d) = x³
```

### 步骤二：转化为理想（Ideal）等式
为了利用 Dedekind 环的性质，将上述元素等式转化为由它们生成的理想（Ideals）的等式。
设：
*   `I = ⟨y + √d⟩`
*   `J = ⟨y - √d⟩`
*   `L = ⟨x⟩`

于是得到理想的乘积等式：
```text
I · J = L³
```

### 步骤三：证明理想互素并应用“下降法”
在给定的前提条件下，作者证明了方程的任何解都必然使得 `x` 为奇数，且 `y` 和 `d` 互素。这可以推导出理想 `I` 和 `J` 是**互素**的。

在 Dedekind 环（此处为 `D`）中，有一个重要的下降法性质：**如果两个互素理想的乘积是一个理想的 n 次方，那么这两个理想各自也必然是某个理想的 n 次方。**
因此，存在 `D` 中的某个理想 `L₁`，使得：
```text
I = L₁³
```

### 步骤四：利用类数条件得出主理想
这是连接类群计算的关键一步。
由定义可知，`I = ⟨y + √d⟩` 是由单个元素生成的，因此 `I` 是一个**主理想（Principal Ideal）**。
现在我们有 `L₁³` 是主理想，并且前提条件规定了类数 `h(D)` 不能被 3 整除。
根据类群的群论性质（基于拉格朗日定理）：如果 `gcd(n, h(D)) = 1` 且 `Iⁿ` 是主理想，则 `I` 本身也必须是主理想。
因此得出结论：**`L₁` 也是主理想**。

### 步骤五：展开并比较系数
既然 `L₁` 是主理想，它可以由 `D` 中的某个元素 `α = a + b√d`（其中 `a, b ∈ ℤ`）生成，即 `L₁ = ⟨α⟩`。
代回理想等式 `I = L₁³`，意味着它们的生成元之间只相差一个单位（unit）`u`：
```text
y + √d = u · α³
```

在论文考虑的负 `d` 范围内，环 `D` 的单位群非常小，所有的单位都是立方数，所以可以不失一般性地假设 `u = 1`。
将 `α = a + b√d` 代入并展开立方：
```text
y + √d = (a + b√d)³ = a(a² + 3b²d) + b(3a² + b²d)√d
```

比较等式两边 `√d` 的系数，得到一个关于整数的方程：
```text
1 = b(3a² + b²d)
```

### 步骤六：在整数范围内得出最终解
因为 `a, b, d` 都是整数，要使两个整数的乘积为 1，`b` 只能取 `±1`：
*   **若 `b = 1`**：`1 = 3a² + d  =>  d = 1 - 3a²`
*   **若 `b = -1`**：`-1 = 3a² + d  =>  d = -1 - 3a²`

综合这两种情况，只有当给定的 `d` 满足：
```text
d = -3a² ± 1
```
对于某个整数 `a` 成立时，方程才可能有解。

接着，比较实数部分的系数来求解 `y`：
```text
y = a(a² + 3b²d)
```
将 `b = ±1` 代入，得到最终 `y` 的通解公式：
```text
y = ±a(3d + a²)
```

## 3. 具体应用案例

论文在前面章节辛辛苦苦计算出了几个特定 `d` 值的类数，发现 `d ∈ {-1, -2, -5, -6, -13}` 时的类数分别为 `1, 1, 2, 2, 2`。**这些类数都不被 3 整除**，完美满足了求解前提，因此可以直接套用上述结论。

### 有解的情况
1. **对于 `d = -1`** (`y² = x³ - 1`)：
   * 检查条件：可以写成 `d = 1 - 3(0)²`，此时 `a = 0`。
   * 求解：代入公式得 `y = 0`。唯一解为 `(x, y) = (1, 0)`。
2. **对于 `d = -2`** (`y² = x³ - 2`)：
   * 检查条件：可以写成 `d = 1 - 3(1)²`，此时 `a = 1`。
   * 求解：代入公式得 `y = ±1(3(-2) + 1) = ±5`。解为 `(x, y) = (3, ±5)`。（这正是费马当年提出的结论）
3. **对于 `d = -13`** (`y² = x³ - 13`)：
   * 检查条件：可以写成 `d = -1 - 3(2)²`，此时 `a = 2`。
   * 求解：代入公式得 `y = ±2(3(-13) + 4) = ±70`。解为 `(x, y) = (17, ±70)`。

### 无解的情况
* **对于 `d = -5` 和 `d = -6`**：
  * 检查条件：找不到任何整数 `a` 能满足 `d = -3a² ± 1`。
  * 结论：这两个方程**没有整数解**。

---

## 总结

该方法展示了代数数论的强大威力：将原本在整数域 `ℤ` 中极其困难的非线性丢番图方程，提升到二次域 `ℤ[√d]` 中。通过利用**理想的唯一分解**和**类群性质**，巧妙地将高次方程转化为求解一个简单的一元二次多项式 `d = -3a² ± 1`，从而干净利落地显式求出了方程的最终解。
